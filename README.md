# 41 COMPUTER Solution & Technology

Website statis hasil migrasi dari tema Blogger untuk GitHub Pages.

## Cara menerbitkan

1. Unggah seluruh isi folder ini ke root repository `MaFiNas/41-computer-solution`.
2. Buka **Settings → Pages**.
3. Pilih **Deploy from a branch**.
4. Pilih branch **main** dan folder **/(root)**, lalu **Save**.
5. Situs akan tersedia di `https://mafinas.github.io/41-computer-solution/`.

## Struktur

- `index.html` — halaman utama
- `assets/css/style.css` — seluruh tampilan
- `assets/js/main.js` — fitur interaktif
- `404.html` — halaman kesalahan
- `robots.txt` dan `sitemap.xml` — SEO
- `manifest.json` — metadata aplikasi web

## Catatan

Form yang terhubung ke Google Apps Script tetap memakai URL backend yang tersimpan di kode sumber. Uji booking dan tracking setelah situs aktif.
